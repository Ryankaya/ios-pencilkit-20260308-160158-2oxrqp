# PencilKit Sketch Pad

A minimal SwiftUI iOS app that demonstrates **PencilKit** by embedding `PKCanvasView` inside SwiftUI, switching inking tools, adjusting stroke width, and clearing the drawing.

## Feature Demonstrated

- `PencilKit` canvas integration with SwiftUI (`UIViewRepresentable`)
- Dynamic `PKInkingTool` switching (Pen, Pencil, Marker)
- Live stroke-width control

## Apple Documentation Links

- https://developer.apple.com/documentation/pencilkit
- https://developer.apple.com/documentation/pencilkit/pkcanvasview
- https://developer.apple.com/documentation/pencilkit/pkinkingtool

## Run

1. `xcodegen generate`
2. Open the generated `.xcodeproj` and run on iOS 17+ simulator/device.
